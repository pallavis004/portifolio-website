Data Science & ML

Placement Prediction & Analysis

Predictive Modeling & Statistical Analysis

Project Summary

Campus placements are critical milestones for universities and students. This project provides a machine learning approach to analyze academic performance, project experience, and technical skill scores to predict whether a student will secure a placement and what salary tier they fall into.

Key Features

  • Statistical Analysis: Identifying correlations between academic CGPA, active backlogs, extra-curriculars, and employment ratios.
  • Supervised Learning: Training Classification algorithms (Decision Trees, Random Forest, and Support Vector Machines) to find optimal models with high accuracy.
  • User Dashboard: Visual representation of predictions using clear charts, highlighting fields where students can improve.

Python Jupyter Notebook Pandas & NumPy Scikit-Learn Seaborn

Cybersecurity & AI

Sentinel-AI

Human-Centric Security Companion

Project Summary

Sentinel-AI bridges the gap between advanced cybersecurity log audits and end-user comprehension. Powered by intelligent model heuristics, it watches for threat patterns and formats alerts in accessible plain-text, helping users remediate vulnerabilities before they are exploited.

Key Features

  • Threat Pattern Detection: Parses system activities and logs to spot anomalous network probes, login failures, or privilege changes.
  • Conversational Advisor: An LLM-powered interactive assistant suggesting immediate commands to secure firewall setups, revoke unauthorized keys, or close ports.
  • Vulnerability Scorecard: Generates real-time hazard ratings based on security compliance benchmarks.

Tech Stack Used

LLM APIs Python Backend Network Log Parsers JSON Web Tokens
