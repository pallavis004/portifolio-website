Placement Prediction & Analysis

A machine learning-based system that predicts whether a student is likely to be placed based on academic performance, skills, and other relevant factors — helping students understand their placement readiness and enabling institutions to make data-driven decisions.

Objectives

  • Predict student placement outcomes using historical placement data
  • Identify factors that influence placement success
  • Assist students in improving employability skills
  • Support colleges in data-driven training and placement decisions

Dataset Features

  • Academic scores (10th, 12th, CGPA)
  • Degree specialization
  • Communication & technical skills
  • Internship experience & certifications
  • Placement status (Placed / Not Placed)

ML Algorithms Used

Logistic Regression Decision Tree Random Forest Support Vector Machine (SVM)

Evaluation Metrics

Accuracy Precision Recall F1-Score Confusion Matrix

Key Insights

  • Identifies key academic and skill factors affecting placements
  • Reveals the relationship between CGPA and placement probability
  • Measures the impact of internships and certifications on employability
  • Uncovers trends and patterns in historical placement data

Methodology

  1. Data Collection
  2. Preprocessing — missing values, encoding, feature scaling
  3. Exploratory Data Analysis (EDA)
  4. Model Training & Evaluation
  5. Prediction and Result Analysis

Future Enhancements

  • Integration with real-time student databases
  • Personalized placement recommendations
  • Resume analysis and skill-gap detection
  • Job role recommendation system
  • Web-based dashboard for visualization and reporting

Sentinel-AI

Human-Centric Cybersecurity Companion Powered by AI

An intelligent cybersecurity companion designed to help individuals and organizations stay safe in the digital world. Unlike traditional security tools, Sentinel-AI takes a human-centric approach — providing clear guidance, personalized recommendations, and real-time assistance to bridge the gap between complex threats and everyday users.

Problem Statement

Cybersecurity threats such as phishing attacks, malware, identity theft, and online scams are becoming increasingly sophisticated. Many users lack the technical knowledge to identify and respond effectively. Sentinel-AI acts as a trusted digital security assistant that helps users make informed, confident security decisions.

Objectives

  • Detect potential cybersecurity threats in real time
  • Provide users with understandable security recommendations
  • Identify phishing emails, malicious links, and suspicious activities
  • Improve cybersecurity awareness through interactive guidance
  • Reduce the risk of human error in security-related decisions

Key Features

AI-Powered Threat Detection Identifies suspicious emails, websites, files, and network activities using intelligent algorithms.
Phishing Detection Analyzes URLs, email content, and sender information to detect phishing attempts before users become victims.
Security Decision Assistant Provides step-by-step recommendations when users encounter suspicious messages or software downloads.
Personalized Risk Assessment Evaluates user behavior and security practices to generate personalized risk scores and improvement suggestions.
Cybersecurity Education Offers simple explanations, security tips, and awareness training tailored to the user's knowledge level.
Real-Time Alerts Notifies users immediately when potential threats or unusual activities are detected.

System Workflow

  1. User interacts with digital content — emails, websites, or files
  2. Sentinel-AI analyzes content for potential security threats
  3. System identifies suspicious activities and evaluates risk levels
  4. Users receive alerts and clear recommendations
  5. Actionable guidance helps users respond safely
  6. Security insights are recorded to improve future threat detection

Benefits

  • Makes cybersecurity accessible to non-technical users
  • Enhances awareness and security decision-making
  • Provides proactive protection against cyber threats
  • Reduces the likelihood of successful phishing and scam attacks
  • Builds user confidence in handling digital security challenges

Future Enhancements
